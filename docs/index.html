Frame Update Order

1 — Clear Buffers
glClear(COLOR | DEPTH) · calculate deltaTime
2 — Input
player.processKeyboardMouse() · WASD + mouse look + shoot flag
3 — Render World
world.Draw() → Skybox → Ground → Trees (instanced) → Decorations
4 — Render Player
player.controlPlayerRendering() → gun model + recoil animation + muzzle flash
5 — Update & Render Enemies
manager.manage() → spawn · chase · attack · explosion anim · audio
6 — Collision Detection
detector.Detect() → bullet↔enemy AABB · laser↔player AABB
7 — HUD Overlay
hud.inGameScreen() → health bar + kill counter (orthographic text pass)
8 — Swap Buffers
glfwSwapBuffers() · glfwPollEvents()

Environment
include/world.h · src/world.cpp
Owns all environment rendering: terrain, skybox, trees, and decorations. Selects assets based on environment type.

Physics
include/collision_detection.h · src/collision_detection.cpp
Runs all ray–box intersection tests once per frame, updating player health and enemy death state.
  • If player shot: Ray(player.pos, player.front) vs each enemy AABB
  • Each active enemy laser Ray vs player AABB

  • Resets canDamage flag after laser lands

GLSL Shaders

Shader Programs

All shaders live in shaders/. Each program = one row of stage badges.

VERTFRAG
model
Standard 3D rendering. Transforms vertices by MVP matrices and samples a diffuse texture.
Used by: Model (gun, drone, decorations)
VERTGEOMFRAG
explode
Geometry shader that translates triangles along their face normal proportional to a time uniform: creates the enemy death burst.
Used by: Enemy (dying animation)
VERTFRAG
ground
Renders the terrain plane. Tiles a ground texture across the 30×30 unit quad.
Used by: World (terrain)
VERTFRAG
skybox
Samples a cubemap using the vertex position as a direction vector. Supports a grayscale uniform for the death-screen effect.
Used by: SkyBox
VERTFRAG
text
2D screen-space shader for FreeType glyph rendering. Reads the red channel as alpha and tints with a color uniform.
Used by: TextRenderer
FRAG
laser
Fragment-only override that outputs a solid red color, giving enemy laser beams their appearance regardless of underlying geometry UV.
Used by: Enemy (laser beam model)
VERT
instancing
Vertex stage for instanced tree/decoration rendering. Reads a per-instance model matrix from a vertex attribute buffer instead of a uniform.
Used by: World (trees + surroundings). Pairs with model.frag
External Libraries

Dependencies

All vendor headers live in include/. System libraries are linked via CMake.

GLAD
OpenGL 3.3 core function pointer loader. Generated loader included directly in the project.
Used by: Shader, Mesh, glfw_setup
GLFW 3
Cross-platform window creation, OpenGL context, keyboard/mouse input, and event loop.
Used by: glfw_setup, Player (input)
GLM
Header-only vector and matrix math library mirroring GLSL types (vec2/3/4, mat4, etc.).
Used by: almost all systems
ASSIMP
3D model importer supporting OBJ, FBX, and many other formats. Processes mesh hierarchy and materials.
Used by: Model
STB Image
Single-header image loader for PNG, JPEG, and other formats. Used to upload textures to the GPU.
Used by: Model, texture_loading
FreeType 2
Font rendering library. Rasterizes .ttf glyphs into bitmaps that are uploaded as GL textures for text rendering.
Used by: TextRenderer
miniaudio
Single-header, cross-platform audio engine. Plays and loops .wav files for all in-game sounds.
Used by: Player, Enemy
